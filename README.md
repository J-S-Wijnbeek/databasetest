# databasetest

This repository contains SQL scripts for managing a music database with artists and songs.

## Files

- **`script 1.sql`** - Main database schema and initial data
- **`top2000_all_artists_songs 1 (1).sql`** - Complete dataset of artists and songs
- **`top2000_all_entries 1 (1).sql`** - All entries data
- **`update_artist_biography_photo.sql`** - SQL queries to update artist biographies and photos
- **`UPDATE_ARTIST_GUIDE.md`** - Detailed guide on how to use the update queries

## Quick Start - Updating Artist Biographies and Photos

To fill in the `Biography` and `Photo` fields for artists:

1. Review the **`UPDATE_ARTIST_GUIDE.md`** for detailed instructions
2. Open **`update_artist_biography_photo.sql`** in your SQL client
3. Choose the appropriate update method for your needs
4. Execute the queries against your database

See [`UPDATE_ARTIST_GUIDE.md`](./UPDATE_ARTIST_GUIDE.md) for complete documentation.
